# � 로또 추첨기 (Lottery Number Generator)

> 행운의 숫자를 찾고 계신가요? 아름다운 애니메이션과 함께 로또 번호를 뽑아보세요! 🍀

## 📸 프로젝트 체험하기

https://yonghwan1.netlify.app/

## 🌟 프로젝트 소개

이 프로젝트는 한국 로또 6/45 게임을 위한 현대적이고 아름다운 번호 생성기입니다. 단순한 기능을 넘어서, 사용자에게 즐거운 경험을 제공하는 인터랙티브 웹 애플리케이션입니다.

### ✨ 왜 이 프로젝트를 만들었나요?

- 기존의 단조로운 번호 생성기에서 벗어나 **시각적 즐거움**을 제공
- **실제 로또 공의 색상 규칙**을 적용하여 현실감 증대
- **부드러운 애니메이션**으로 추첨 과정의 재미 극대화

## 🎯 주요 기능

### 🎲 핵심 기능

- **스마트 번호 생성**: 1~45 범위에서 중복 없는 6개 번호 자동 생성
- **실시간 애니메이션**: 번호가 순차적으로 나타나는 부드러운 애니메이션
- **공식 색상 적용**: 실제 로또 번호별 색상 규칙 준수
- **자동 정렬**: 생성된 번호를 오름차순으로 자동 정렬
- **중복 추첨 방지**: 추첨 중 버튼 비활성화로 안정성 보장

### 🎨 디자인 특징

- **현대적 UI/UX**: 글래스모피즘과 그라디언트 디자인
- **3D 로또볼**: 실감나는 3D 효과와 하이라이트
- **반응형 디자인**: 데스크톱부터 모바일까지 완벽 대응
- **접근성 고려**: 색상 대비와 폰트 가독성 최적화

## 🎨 로또볼 색상 시스템

실제 한국 로또와 동일한 색상 규칙을 적용했습니다:

| 번호 범위 | 색상      | 설명                        |
| --------- | --------- | --------------------------- |
| 1 ~ 10    | 🟡 노란색 | 따뜻하고 밝은 느낌          |
| 11 ~ 20   | 🔵 파란색 | 시원하고 신뢰감 있는 색상   |
| 21 ~ 30   | 🔴 빨간색 | 열정적이고 강렬한 색상      |
| 31 ~ 40   | ⚫ 검은색 | 고급스럽고 세련된 색상      |
| 41 ~ 45   | 🟢 초록색 | 자연스럽고 안정감 있는 색상 |

## 🔧 핵심 알고리즘 설명

### 1. 중복 없는 번호 생성

```javascript
// Set 자료구조를 활용한 효율적인 중복 제거
const numbers = [];
while (numbers.length < 6) {
  const num = Math.floor(Math.random() * 45) + 1;
  if (!numbers.includes(num)) {
    numbers.push(num);
  }
}
```

### 2. 순차 애니메이션

```javascript
// setTimeout을 활용한 연쇄 애니메이션
numbers.forEach((number, index) => {
  setTimeout(() => {
    // 각 볼이 200ms 간격으로 나타남
    showBall(number, index);
  }, index * 200);
});
```
