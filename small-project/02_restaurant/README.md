## 💡 주요 기능 설명

### ✨ 랜덤 식당 추천

- 여러 식당 정보를 미리 저장해두고, 추천 버튼을 누르면 무작위로 식당을 보여줍니다.

### 📍 위치 기반 추천 (설계)

- 사용자의 위치 정보를 받아, 가까운 식당을 우선적으로 추천하는 기능을 추가할 예정입니다.

### 🍽️ 사용자 선호도 반영 (설계)

- 사용자가 좋아하는 음식, 싫어하는 음식 정보를 입력하면, 그에 맞는 식당을 더 똑똑하게 추천할 수 있도록 발전시킬 계획입니다.
